Global Trade Map

The Global Trade Mapâ„¢ (GTD) covers 220 countries and territories and 5300 products of the Harmonized System. The monthly, quarterly and yearly trade flows are available from the most aggregated level to the tariff line level.

Data Collection

  • The data is scrapped from the website using Selenium, FY 2003-2023 for 120 countries which had at least 1bn$ import value in 2021-2022, for top 100 product categories
  • The map is based on import values, because import data is fuller: e.g. there's no trade data FY 2022 for Russia, UAE, etc. But their export data can be collected through import data of their counterparts

Missing entries

  • Several major exporters are missing 2022 trade data at the date of collection (July 28th, 2023). Many smaller traders or countries with political instability (e.g. Venezuela) are missing trade data for the past several years. To compensate for the gaps, I've used import data to use the data from their counterparts
  • In case the selected year data is missing, the map retrives the previous year data for a country. If the previos year data is missing as well, the country's tradeflows do not appear in the map
